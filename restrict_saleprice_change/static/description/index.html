Sale Price Change Restriction

This app is a tool to restrict User to change the Unit Price of the product on sales order for certain User once the Price list is applied.

Easily grant user rights to change the unit price on Sales order. Just check 'Manage Sales Price Change' inside user Other Extra Rights settings (Odoo developer mode is required)
When an person not having this right tries to change the unit price of product if the pricelist is applied on sale order than an warning is Raised.
Users outside the group will not be able change the Price of product to less than the Unit Price in Sale Order.

Prerequisites before Installation

User needs to Configure the Pricelist in the settings.Only when the Pricelist is selected on the Sale Order the restriction will take Place.

Configurations

After Installation of the module the user needs to give the rights to the user in the Settings to Manage the Price Change.
